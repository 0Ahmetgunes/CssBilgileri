# Temel CSS Konuları
- CSS Nedir?

- CSS'in HTML ile Kullanımı (Inline, Internal, External CSS)

- Seçiciler (Selectors)
  - Evrensel (*)
  - Tür (Element) (h1, p, div)
  - Sınıf (.class)
  - ID (#id)
  - Grup (h1, p, div)
  - Bağlantı (Attribute) Seçicileri

- CSS Kutu Modeli (Box Model)
  - Padding

  - Border
  - Content

- Renkler ve Arka Planlar
  - Renk tanımlama (RGB, HEX, HSL)
  - Opaklık (opacity)

- Metin ve Yazı Tipleri (Typography)
  - Font ailesi (font-family)
  - Font boyutu (font-size)
  - Kalınlık (font-weight)
  - Satır yüksekliği (line-height)

- Bağlantılar ve Düğmeler

# Orta Seviye CSS Konuları

- Pozisyonlandırma (Positioning)
  - Statik (static)
  - Göreli (relative)
  - Mutlak (absolute)
  - Sabit (fixed)
  - Yapışkan (sticky)

- Ekran Görüntüleme (Display) ve Görünürlük (Visibility)
  - block, inline, inline-block, none
  - Görünürlük kontrolü (visibility, opacity)

- Float ve Clear Kullanımı

- Esneklik ve Düzeni (Flexbox)
  - display: flex
  - justify-content
  - align-items
  - flex-wrap

- Grid Sistemi (CSS Grid)
  - Satır ve sütun yapıları
  - grid-template-rows ve grid-template-columns

- Medya Sorguları (Media Queries) ve Duyarlı Tasarım (Responsive Design)

- Transform ve Animasyonlar
  - transform: rotate, scale, translate
  - transition
  - animation

- Gelişmiş Seçiciler
  - nth-child
  - first-child, last-child
  - not()

- CSS Değişkenleri (Custom Properties - Variables)
